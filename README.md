
⭐ Feedback Platform
📌 Project Overview

The Feedback Platform is a web-based application designed to collect and manage customer feedback for products and services. The system allows companies to gather valuable insights from users while rewarding customers for sharing their experiences.

The platform creates an interactive environment where companies can display their products, collect user feedback, analyze responses, and improve product quality based on customer insights.

This project was developed as part of the Undergraduate (Bachelor of IT) Final Year Project.

🎯 Project Objectives

• Create a centralized platform to collect product feedback
• Enable users to submit reviews and suggestions
• Help companies analyze customer satisfaction
• Provide rewards to users for contributing feedback
• Improve product quality through customer insights

⚙️ System Features
User Features

User registration and login

Browse products

Submit feedback and ratings

Earn rewards for feedback submission

Company / Admin Features

Add and manage products

View customer feedback

Analyze product reviews

Monitor user engagement

🏗 System Architecture

The system consists of two main components:

Frontend
Handles user interaction and interface.

Backend
Processes user requests, manages feedback data, and communicates with the database.
